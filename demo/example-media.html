Media Node Examples

This document demonstrates how ox-lexical handles media nodes. These example files are small enough to commit to the repository.

Video

Video uses the #+BEGIN_VIDEO special block:

Org Source

#+BEGIN_VIDEO :src example-video.mp4

#+END_VIDEO

Generation Commands

These example files were generated with: 

# Audio: 1-second 440Hz tone
ffmpeg -f lavfi -i "sine=frequency=440:duration=1" \
  -c:a libmp3lame -q:a 9 example-audio.mp3

# Image: 200x100 gradient with text
magick -size 200x100 "gradient:#667eea-#764ba2" \
  -font Helvetica -pointsize 16 -fill white \
  -gravity center -annotate +0+0 "ox-lexical" example-image.png

# Video: 2-second 160x90 with text overlay
ffmpeg -f lavfi -i "color=c=#667eea:s=160x90:d=2" \
  -vf "drawtext=text='ox-lexical':fontsize=14:fontcolor=white:x=(w-tw)/2:y=(h-th)/2" \
  -c:v libx264 -preset ultrafast -crf 28 -pix_fmt yuv420p example-video.mp4

Total size: ~12KB - small enough to commit to the repository.
