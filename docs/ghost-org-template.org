** Block-Level Elements

*** Paragraphs

Regular text creates paragraph nodes. Multiple paragraphs are separate nodes.

A blank line between text creates a new paragraph.

*** Headings

Org headings (=*=, =**=, =***=) map to Ghost =heading= nodes with tags h2-h6.
(The document title becomes h1 via Ghost metadata, not in the Lexical content.)

*** Lists

Unordered list (=list= node with =listType: "bullet"=):
- First item (=listitem= node)
- Second item
- Third item

Ordered list (=listType: "number"=):
1. First item
2. Second item
3. Third item

Nested lists (=listitem= contains child =list=):
- Parent item
  - Nested item
  - Another nested
- Back to parent

#+BEGIN_ASIDE
This maps to an =aside= node.

#+END_ASIDE

** Inline Elements

*** Text Formatting

These map to =text= nodes with format bitmask:

- *Bold text* (format: 1)
- /Italic text/ (format: 2)
- +Strikethrough+ (format: 4)
- _Underlined_ (format: 8)
- =Inline code= (format: 16)
- *_Bold and underline_* (format: 9 = 1+8)
- */Bold italic/* (format: 3 = 1+2)

*** Links

Regular links map to =link= nodes containing =text=:

[[https://ghost.org][Ghost Website]]

[[https://example.com]]

*** Line Breaks

Use =\\= for soft line breaks within a paragraph:\\
This stays in the same paragraph but on a new line.

(Note: linebreaks only valid in paragraph and quote blocks)
